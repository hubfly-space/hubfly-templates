# Next.js HubFly Template

This template includes:
- Next.js (App Router)
- API routes (`/api/health`, `/api/echo`)
- Server action host route tester form
- Vitest test suite
- Next.js image optimization sample

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Run tests

```bash
npm test
```

## Build

```bash
npm run build
```
